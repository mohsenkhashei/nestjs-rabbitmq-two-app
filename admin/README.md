## Description

[NestJS](https://github.com/nestjs/nest) & Mysql with RestAPI and RabbitMQ

## Installation

```bash
$ yarn install
```

## Running the app

Before Running this App make sure to run dependencies first

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
